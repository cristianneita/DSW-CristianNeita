Atributos en HTML 5:

Los elementos en HTML tienen atributos; estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios. Estos son algunos atributos:

Nombre del atributo Elementos Descripción
accept < form>, < input> Lista de tipos aceptados por el servidor, usualmente un tipo de archivo
accept-charset < form > Lista de juegos de caracteres soportados.
action < form > La URL del programa que procesa la información enviada en el formulario.
autocomplete < form>, < input>

Indica si los controles en este formulario pueden tener por defecto sus valores completados automáticamente por el navegador.
autofocus < button>, < input>, < keygen>, < select>, < textarea> El elemento debería recibir foco automáticamente después de haberse cargado la página. El elemento debería recibir foco automáticamente después de haberse cargado la página.
autplay < audio>, < video> El audio o video debería empezar lo antes posible.
bgcolor

< body>, < col>, < colgroup>, < marquee>, < table>, < tbody>, < tfoot>, < td>, < th>, < tr>

 Color de fondo del elemento.
border < img>, < object>, < table> El ancho del borde.
checked < command>, < input> Indica que el elemento debería estar marcado al cargar la página.
color < basefont>, < font>, < hr> Este atributo establece el color de texto utilizando un nombre de color o en formato hexadecimal #RRGGBB.
download < a>, < area> Indica que el hipervínculo es utilizado para descargar un recurso.
icon < command> Especifica una imagen relacionada con el comando.
label < track> Especifica el título de la pista con un formato legible por el usuario.
low < meter> Indica el valor mas alto del rango bajo.
max < input>, < meter>, < progress> Indica el máximo valor aceptado.
maxlength < input>, < textarea>
muted < video > Indica si el audio será silenciado inicialmente al cargar la página.
name < button>, < form>, < fieldset>, < iframe>, < input>, < keygen>, < object>, < output>, < select>, < textarea>, < map>, < meta>, < param> Nombre del elemento. Por ejemplo utilizado por el servidor para identificar los campos en el envio de formularios.
radiogroup < command> Indica los nombres de los comandos de tipo radio, que seran alternados si el comando en si es alternado. Este atributo debe ser omitido si el tipo de comando no es radio.
rows < textarea> Define el número de filas en un área de texto
rowspan < td>, < th> Define el número de filas que una celda de una tabla debe abarcar.
size < input>, < select> Define el ancho del elemento (en píxeles). Si el atributo del elemento es del tipo text o password entonces es el número de caracteres.
title Atributo Global Texto a ser mostrado cuando el cursor esté sobre el elemento.
width < canvas>, < embed>, < iframe>, < img>, < input>, < object>, < video> Para los elementos enumerados aquí, esto establece el ancho del elemento.
Colaboradores MDN (23 de marzo del 2019). Referencia de Atributos HTML. Mozilla. https://developer.mozilla.org/es/docs/Web/HTML/Atributos
